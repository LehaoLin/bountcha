## BounTCHA prototype

### How to run the prototype

#### Build frontend

`cd fronend; pnpm i; pnpm build`

#### Run

`python3 main.py`

Open `http://localhost:7777`

### Modify the time bias range

In `main.py`, modify `lower_bound` and `upper_bound`

### To compress the video

convert.py
